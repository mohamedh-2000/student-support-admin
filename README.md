# Student Support Admin

Vite + React + MUI. נתונים נשמרים ב-LocalStorage.

## ריצה
npm i
npm run dev

## זרעי נתונים (seed)
בכניסה למסך הבית נטענים נתוני דמו אם ה-LS ריק.

## ניווט
- /tickets, /tickets/new, /tickets/:id
- /categories, /categories/new
